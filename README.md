# Mini JIRA

A full-stack project management application inspired by JIRA, built with React and Node.js.

## Project Structure

- **backend/** - Node.js/Express backend server
- **frontend/** - React frontend application

## Features

- User authentication and authorization
- Task management with drag-and-drop
- Task status tracking (To Do, In Progress, Done)
- User dashboard and analytics
- Responsive design

## Getting Started

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Technologies Used

### Backend
- Node.js
- Express
- MongoDB
- JWT Authentication

### Frontend
- React
- Redux Toolkit
- Tailwind CSS
- React Beautiful DnD
- Vite
